# Exporting

## Spectrum data

File > Export > CSV… or JCAMP-DX… exports the data of whichever spectrum
is selected in the sidebar (a selection is required; both items are
grayed out otherwise).

## The plot

File > Export > Plot as PNG… or Plot as PDF… saves an image of the plot.
Copy Plot (Shift-Command-C) puts a PNG straight on the clipboard instead
of saving a file. All three render the plot's current view, whatever's
zoomed or panned, on a plain white background regardless of the app's
appearance, so it's ready to drop into a document as-is.

If peak labels are turned on (View > Peak Labels, Shift-Command-L, or the
Labels toggle in the plot toolbar), they're baked into the exported
image the same way they appear on screen.

## Measurements

In the results table, Copy puts a tab-separated copy of the current
selection (or everything, if nothing's selected) on the clipboard, ready
to paste into a spreadsheet. Export CSV… writes the same data to a CSV
file you choose.
